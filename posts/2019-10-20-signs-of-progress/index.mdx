## AWIP

- [Passport](http://www.passportjs.org) 이용해서 로그인 기능.
  - JWT 방식의 경우 Passport가 전혀 도움이 되는 것 같지 않아서 제거할 예정.
- [MongoDB](https://www.mongodb.com), [Mongoose](https://mongoosejs.com) 설정.
- 로그인 상태 서버에서 렌더링: _app.js.
- Facebook 로그인
  - Facebook API 설명을 찾기가 너무 힘들어서 [passport-facebook-token](https://github.com/drudge/passport-facebook-token) 사용.
- Google 로그인
  - [@types/gapi.auth2](https://www.npmjs.com/package/@types/gapi.auth2)
